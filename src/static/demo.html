About This Project

This project contains various word game implementations including Wordle, Semantle, and crossword puzzles. It uses a FastAPI backend for game logic and JWT tokens for stateless session management.

Features

  • Multiple game variants (English and French)
  • Variable word lengths (3-9 letters)
  • Responsive design for mobile and desktop
  • Stateless architecture using JWT tokens
  • Daily word challenges

Running Locally

To run the game with full functionality, you need to start the backend server:

# Install dependencies
uv sync

# Start the server
uv run src/main_riddle_game_server.py your-secret-key

# Access the game at:
# http://127.0.0.1:8000/wordle-en-5
